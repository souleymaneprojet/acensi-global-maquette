# Maquette ACENSI Global - HTML/CSS/JavaScript

## Objectif

Ce projet est une maquette codée de la page d'accueil d'ACENSI Canada, imaginée comme une version globale / mondiale.

L'objectif n'est pas de copier exactement le site existant, mais de proposer :
- une structure claire ;
- un design moderne ;
- un code propre et compréhensible ;
- une bonne ergonomie ;
- une version responsive.

## Technologies utilisées

- HTML5 : structure de la page
- CSS3 : design, responsive, animations
- JavaScript : menu mobile et petites interactions

## Comment exécuter le projet

Aucun serveur n'est nécessaire.

1. Dézipper le dossier.
2. Ouvrir le fichier `index.html` dans le navigateur.
3. Tester la version desktop et mobile.

## Pourquoi HTML plutôt que PHP ?

La demande concerne une maquette de page d'accueil. Le plus important est donc :
- la qualité du front-end ;
- la structure ;
- le responsive ;
- le design ;
- l'expérience utilisateur.

PHP serait utile pour un site dynamique avec base de données, espace admin ou CMS.
Ici, HTML/CSS/JS est plus simple, plus lisible et plus adapté à une maquette.

## Sections de la page

1. Header : navigation claire et bouton de contact.
2. Hero : message principal + identité globale.
3. Chiffres clés : éléments de confiance.
4. À propos : présentation courte.
5. Expertises : cartes lisibles.
6. Réalisations : cas d'usage.
7. Carrières : aspect humain.
8. Contact : conversion utilisateur.
9. Footer : informations finales.

## Choix personnels

J'ai choisi d'imaginer une version mondiale avec :
- une identité "ACENSI Global" ;
- une carte visuelle de présence internationale ;
- des sections plus modernes et plus lisibles ;
- un design rouge/blanc/noir cohérent avec l'esprit ACENSI ;
- une structure simple à maintenir.
